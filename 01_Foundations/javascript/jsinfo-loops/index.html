Printing all prime numbers

  1. Get input number which will the the maximum range of prime number lookup.
  2. Loop through the range -> [1, number].
  3. For every integer in the range, check if the integer is prime.
  4. If the integer is a prime then put it in the outputArray
  5. Output the array

Identifying prime number

Prime numbers are the natural numbers greater than 1 with exactly two factors, i.e. 1 and the number itself.
  1. Get number.
  2. Let range = [1, number]
  3. Check if the number % integer_in_range == 0
  4. If yes then push it to the factorsArray
  5. Count the elements in factorsArray
  6. If the number of factors is just 2 (1 and itself, then it the number is a prime)
